# 02. 希腊字母实战解读 (The Greeks)

别把希腊字母当成数学公式，把它们当成你驾驶期权这辆赛车的**仪表盘**。

## 1. Delta ($\Delta$)：方向与概率

Delta 衡量的是**股价变动 1 美元，期权价格变动多少**。

- **Call Delta**: 0 到 1。
- **Put Delta**: -1 到 0。
- **实战意义**：
    1.  **对冲比率**：Delta = 0.5，意味着你持有 1 张期权（对应 100 股），相当于持有 50 股正股。
    2.  **胜率近似值**：这是最有用的！**Delta 约为期权到期变为实值（ITM）的概率**。
        - 卖出 Delta 0.2 的 Put，意味着你有大约 80% 的概率赚到全额权利金。

## 2. Theta ($\Theta$)：时间的租金

Theta 衡量的是**时间过去一天，期权价格损耗多少**。

- **买方 Theta 是负的**：每天都在失血。
- **卖方 Theta 是正的**：每天都在收租。
- **加速效应**：期权越临近到期日，Theta 损耗越快（尤其是平值期权 ATM）。这也是为什么卖方喜欢做短期的原因之一。

## 3. Vega ($\nu$)：情绪的泡沫

Vega 衡量的是**隐含波动率（IV）变化 1%，期权价格变动多少**。

- **实战口诀**：**IV 高时卖，IV 低时买**。
- 就像弹簧，IV 总是会回归均值。
    - 财报前 IV 暴涨 -> Vega 让期权变贵 -> 此时适合做卖方（Short Vega）。
    - 财报后 IV 暴跌 -> Vega 让期权缩水 -> 买方此时会遭受 "IV Crush"。

> **Q: 什么是隐含波动率 (IV)？**
>
> IV 是**市场对未来波动的预期**（未来的天气预报）。
> - 它是通过期权市场价格**倒推**出来的。期权卖得越贵，IV 就越高。
> - **怎么看**：在交易软件的期权链 (Option Chain) 上直接查看。关注 **IV Rank (IVR)**，IVR > 50% 适合做卖方，IVR < 20% 适合做买方。

## 4. Gamma ($\gamma$)：加速度

Gamma 是 Delta 的变化率。
- **Gamma Risk**：临近到期时，Gamma 最大。股价微小的波动会让 Delta 剧烈跳动，导致你的盈亏坐过山车。
- **建议**：新手卖方尽量在到期前关闭仓位（例如剩 21 天时），避免“Gamma 风险”。

---

## 仪表盘总结

| 字母 | 含义 | 对买方 (Long) | 对卖方 (Short) |
| :--- | :--- | :--- | :--- |
| **Delta** | 方向 | 正相关 | 负相关 |
| **Theta** | 时间 | **敌人** (亏钱) | **朋友** (赚钱) |
| **Vega** | 波动 | 朋友 (希望IV涨) | 敌人 (希望IV跌) |
