# 04. 核心策略体系 (The Core Strategies)

这里我们只介绍经过时间检验、适合散户长期生存的策略。

## 一、 轮动策略 (The Wheel Strategy)

这是美股期权圈最经典的**现金流**策略。它结合了 CSP 和 CC，形成一个完美的闭环。

### 循环逻辑

1.  **第一步：Sell Cash Secured Put (CSP)**
    - 选择你愿意长期持有的好公司（如 AAPL, MSFT）。
    - 卖出 OTM (虚值) Put。
    - **结果 A**：股价没跌破行权价 -> 赚取权利金 -> **重复第一步**。
    - **结果 B**：股价跌破行权价 -> 被行权接盘股票 -> 你的持仓成本 = 行权价 - 权利金 -> **进入第二步**。

2.  **第二步：Sell Covered Call (CC)**
    - 你现在持有 100 股正股。
    - 卖出 OTM (虚值) Call（行权价通常高于你的成本价）。
    - **结果 A**：股价没涨破行权价 -> 赚取权利金 + 股息 -> **重复第二步**。
    - **结果 B**：股价涨破行权价 -> 股票被召回卖出 -> 赚取股票价差 + 权利金 -> **回到第一步**。

### 优点
- 始终在赚取时间价值（Theta）。
- 即使被套，也是套在好公司上，且权利金不断降低成本。

## 二、 垂直价差 (Vertical Spreads)

如果你资金量小，或者不想接盘股票，只想做方向投机，**不要裸买 Call/Put**，请用价差策略。

### 1. 牛市价差 (Bull Call Spread) - 看涨但控制成本
- **操作**：买入低行权价 Call (Buy Leg) + 卖出高行权价 Call (Sell Leg)。
- **逻辑**：卖出的 Call 帮你分摊了买入 Call 的成本（和 Theta 损耗）。
- **特点**：风险有限（最大亏损=权利金差价），收益有限（最大收益=行权价差价 - 成本）。

> **Q: 这里的 Leg (腿) 是什么意思？**
>
> **Leg (腿)** 指的是复杂策略中的**单笔交易**。
> - **Buy Leg**：负责买入的那张期权。
> - **Sell Leg**：负责卖出的那张期权。
> - 就像走路需要两条腿配合，多腿策略利用不同 Leg 的特性（如一买一卖）来构建特殊的风险收益曲线。

### 2. 熊市价差 (Bear Put Spread) - 看跌但控制成本
- **操作**：买入高行权价 Put + 卖出低行权价 Put。
- **逻辑**：同上，降低做空成本。

---

## 三、 铁鹰策略 (Iron Condor) - 赚取横盘收益

当市场无聊、横盘震荡时使用。

- **操作**：卖出宽跨式组合 (Strangle) + 买入更远期的保护腿。
    - 实际上就是：一个熊市价差 (Call Spread) + 一个牛市价差 (Put Spread)。
- **逻辑**：只要股价在一定区间内波动，你就能吃到所有的权利金。
- **风险**：如果发生单边暴涨或暴跌，会有亏损（但因为有保护腿，亏损有限）。
