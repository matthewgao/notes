**永远不要裸卖 Call (Naked Call)**
**做期权的卖方（Seller/Writer）胜率远高于买方（Buyer）**

你不需要自己算，所有行情软件都会直接显示。
- **期权链 (Option Chain)**：在每一行期权报价旁边，通常都有一个 `Imp Vol` 或 `IV` 的列。
    - 例如：AAPL $180 Call 的 IV 可能是 20%。
- **个股详情页**：通常会有 `IV Rank` (IVR) 或 `IV Percentile` (IVP)，告诉你当前的 IV 相对于过去一年是高还是低。
    - **IV Rank > 50%**：说明现在波动率很高，适合**卖**期权。
    - **IV Rank < 20%**：说明现在波动率很低，适合**买**期权。

实战口诀：IV 高时卖，IV 低时买。
就像弹簧，IV 总是会回归均值。
财报前 IV 暴涨 -> Vega 让期权变贵 -> 此时适合做卖方（Short Vega）。
财报后 IV 暴跌 -> Vega 让期权缩水 -> 买方此时会遭受 "IV Crush"。

卖put实际上是 我给别人一个看跌的人低价买如的机会，博弈的是价格是不是比我这个更低，更低的话我亏钱，对方赚更多，如果没到我设定的价格，我赚权益金，对方如果行权呢？


Delta = 0.5，意味着你持有 1 张期权（对应 100 股），相当于持有 50 股正股。
Theta 衡量的是**时间过去一天，期权价格损耗多少**。
Vega 衡量的是**隐含波动率（IV）变化 1%，期权价格变动多少**。
关注 **IV Rank (IVR)**，IVR > 50% 适合做卖方，IVR < 20% 适合做买方。

Gamma 是 Delta 的变化率。
- **Gamma Risk**：临近到期时，Gamma 最大。股价微小的波动会让 Delta 剧烈跳动，导致你的盈亏坐过山车。
- **建议**：新手卖方尽量在到期前关闭仓位（例如剩 21 天时），避免“Gamma 风险”。
