# 05. 实战案例复盘 (Case Studies)

这里我们用真实的美股场景来演示策略的应用。

## 案例一：英伟达 (NVDA) - 波动率收割与低价接盘
*场景：NVDA 刚发布财报，股价从 900 跌到 800，你觉得长期看好，想在 750 接盘。*

- **错误操作**：直接挂单 $750 等待买入。
    - 缺点：如果没跌到 750，你什么都得不到，干等。
- **高手操作**：卖出 NVDA $750 Put (Sell CSP)，到期日选择 2 周后。
    - 假设权利金为 $15/股。
    - **结局 A (股价跌破 750)**：你需要以 750 买入。实际成本 = 750 - 15 = 735。比挂单更便宜！
    - **结局 B (股价没跌破)**：你白赚 $1500 (1张合约)。年化收益率极高。
    - **逻辑**：利用 NVDA 的高波动率 (High IV)，赚取昂贵的权利金。

## 案例二：苹果 (AAPL) - 养老股的增强收益 (Wheel Strategy)
*场景：你持有 100 股 AAPL，成本 170，当前股价 175。你想长期持有，但觉得涨得太慢。*

- **操作**：卖出行权价 185 的 Covered Call，到期日 1 个月后。
    - 假设权利金 $3/股。
- **分析**：
    - 如果 AAPL 涨到 180：你依然持有股票，外加 $300 零花钱。
    - 如果 AAPL 暴涨到 190：你的股票在 185 被卖出。获利 = (185 - 170)股票价差 + 3 权利金 = $18/股。
        - 虽然少赚了 (190-185) 的部分，但这一笔交易锁定了 10% 的利润。
- **心态**：如果被召回，就当作止盈；如果不被召回，就是降低持仓成本。

## 案例三：特斯拉 (TSLA) - 为疯狂的波动买保险
*场景：你重仓 TSLA，成本 200，现在 250。明天是财报日，你怕暴跌，但又不想卖出股票错过暴涨。*

- **操作**：买入 Protective Put (保护性看跌期权)。
    - 买入行权价 240 的 Put (Buy Put)。
- **逻辑**：
    - **如果财报暴雷，跌到 200**：股票亏了，但 Put 会大涨，抵消大部分亏损。你有权按 240 卖出。
    - **如果财报大好，涨到 300**：股票大赚，Put 归零（变成废纸）。
    - **总结**：这相当于支付一笔“保险费”，锁定了最大回撤，同时保留了上涨空间。

> **Q: 暴跌时为什么 Put 会大涨？我必须行权吗？**
>
> 1.  **为什么涨**：暴跌导致 Put 变成**实值 (Intrinsic Value 增加)**，且恐慌导致 **IV 飙升 (Vega 增加)**，双重驱动价格暴涨。
> 2.  **必须行权吗**：**不需要。** 你可以直接把这张暴涨的 Put **卖掉平仓 (Sell to Close)** 换取现金来抵消股票亏损，而不需要真的卖出股票。
